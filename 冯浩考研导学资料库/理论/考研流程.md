# 考研流程篇



## 考研的重要时间节点

| 时间                    | 事件                         |
| ----------------------- | ---------------------------- |
| 2月中旬                 | 考研成绩公布                 |
| 2月下旬-3月上旬         | 考研34所自主划线公布         |
| 3月中旬                 | 考研国家线公布               |
| **3月-4月**             | **考研院校考研复试和调剂**   |
| 6月-7月                 | 录取通知书发放               |
| **3月-10月**            | **考研院校招生简章目录公布** |
| 8月                     | 研招规定发布                 |
| 9月                     | 考研国家考试大纲公布         |
| 九月下旬                | 考研咨询周                   |
| **9月24日 - 9月27日**   | **考研网上预报名（有效的）** |
| **10月10日 - 10月31日** | **考研全国统一网上正式报名** |
| 11月上旬                | 考研现场确认                 |
| 12月                    | 考研准考证打印               |

**报名时间只有一天！！！**不要焦虑，彷徨！！！
